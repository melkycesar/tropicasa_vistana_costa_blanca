Paleta de color de Tropicasa, en orden de proporción de uso:
1. #EF6E21 — color principal de Tropicasa. Usar en acentos, badges o detalles de marca cuando se quiera identificar a la agencia (no para todo el texto).
2. Blanco — el fondo obligatorio detrás del logo, y el color dominante de soporte.
3. #6D6E71 — color secundario, uso minoritario (ej. una línea de texto pequeña, un ícono secundario). Nunca debe competir en protagonismo con el naranja ni con el blanco.
